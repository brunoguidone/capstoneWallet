# capstoneWallet Project

## Explanation

This project runs a demo wallet application in a kubernetes cluster (EKS) with mysql databases in a replicated stateful set.

## Environment and tools

	. AWS (EKS)
	. Terraforms
	. Python
	. Ingress nginx
